# Sobre

No seguinte portfólio irá ficar armazenado projetos para a certificação do curso de Design Responsivo para a Web disponibilizado pela empresa [FreeCodeCamp](https://www.freecodecamp.org/learn/responsive-web-design-v9/).

## Atenção
Os projetos aqui presentes não devem ser usados como exemplo para nenhuma demanda. O foco deste repósitorio é meramente um retrato de meus aprendizados e para que no futuro tenha lembranças nostálgicas.

## Conhecimentos 
### HTML
Como base de qualquer projeto, neste breve curso o HTML foi um dos pilares para desenvolver tabelas, formulários, páginas de blog dentre outros.
<br>
O curso abrange desde o mais teórico como: Qual o papel do HTML na Web, até exemplos práticos como criar um formulário passo a passo, apesar de ser um requisito básico para qualquer programador, acredito que vale a pena estudar um pouco dessa linguagem de marcação, afinal ela está em quase todo site que vemos na internet.
### CSS
Outro pilar do desenvolvimento é o CSS uma linguagem de estilização que não apenas deixa o site com o aspecto mais agradável, mas trás consigo a possibilidade de criar sites mais acessíveis, aumentando fonetes ou modificando cores para que seja mais legível.
